## ✨ Features

- **Multi-Vendor Architecture:** Multiple vendors can register, manage products, and sell on a single platform.
- **Customer Storefront:** Clean and responsive UI for browsing and purchasing.
- **Vendor Dashboard:** Manage products, track orders, and view analytics.
- **Admin Panel:** Control vendors, products, and platform operations.

---

## 🛠️ Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **UI:** Lucide React
- **State Management:** Redux Toolkit
